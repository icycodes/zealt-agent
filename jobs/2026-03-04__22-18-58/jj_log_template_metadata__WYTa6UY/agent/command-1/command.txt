## Your Task

You must run a single `jj log` command that:

1. Targets all non-root, non-working-copy commits in the repository (use the revset `roots(mutable())::@ ~ @` to select committed revisions excluding the current empty working-copy commit).
2. Uses `--no-graph` to suppress the graph decoration.
3. Uses a `--template` (or `-T`) that formats each commit as a single line with the following fields separated by a pipe character (`|`):

   - The author name: `author.name()`
   - The author email: `author.email()`

4. Each record must end with a newline (`\n`).
5. Redirects the output to `/home/user/project/commit_log.txt`.

The template string you must use is:
